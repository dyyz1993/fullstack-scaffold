## Findings

- **CRITICAL BUG**: Content list always shows empty ("暂无内容") despite API returning 7 items
  - Root cause: `ContentListPage.tsx` line 37 checks `Array.isArray(data)` but API returns `{items: [...], count: 7}`
  - API endpoint: GET /api/public/contents returns `{success: true, data: {items: [...], count: 7}}`
  - Code expects `data` to be an array but it's `{items: [], count: N}` object
  - Fix: Change line 37 from `setContents(Array.isArray(data) ? data : [])` to `setContents(Array.isArray(data) ? data : data.items || [])`
