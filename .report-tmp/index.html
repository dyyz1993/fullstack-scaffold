🔧 CLI 工具

命令行工具，提供 RPC 服务和代码生成功能。

帮助信息

biomimic v0.1.0
Biomimic CLI - RPC service & code generation tools

USAGE
  biomimic <command> [options]

COMMANDS
  list                List all notifications
  get                 Get a todo by ID
  create              Create a new notification
  update              Update a todo
  delete              Delete a notification
  unread-count        Get unread notification count
  mark-read           Mark a notification as read
  register            Register a new developer account
  login               Login to get API key
  verify              Verify API key
  config-get          Show current configuration
  config-set          Set configuration value
  config-url          Show or set server URL
  config-status       Check server connection status
  config-reset        Reset configuration to defaults
  config-path         Show config file path

OPTIONS
  --version, -v        Print version
  --help, -h           Print help

示例命令

# 创建待办事项
$ biomimic create --title "Buy groceries" --description "Milk, eggs, bread"

# 查看配置
$ biomimic config-get
{
  "success": true,
  "data": {
    "baseUrl": "http://localhost:3017"
  },
  "tips": []
}

# 用户登录
$ biomimic login --account demo --password demo123

📡 API 接口

⚠️ 网络访问限制

由于 Vite 开发服务器的安全配置，外部网络请求被阻止。API 数据需要在本地环境或通过 VPN 访问。

# API 端点示例 GET /health # 健康检查 GET /api/todos # 获取待办事项列表 POST /api/todos # 创建待办事项 GET /api/notifications # 获取通知列表 POST /api/notifications # 创建通知 GET /api/chat/ws # WebSocket 连接

技术栈

前端: React 18 + TypeScript + Vite + Tailwind CSS 后端: Hono + Node.js + Drizzle ORM 数据库: PostgreSQL 实时通信: WebSocket + SSE 部署: PM2 + Nginx + SSL

项目特性

类型安全 实时通信 模块化架构 CLI 工具 多模板支持 云原生部署 自动生成代码 完整测试覆盖
